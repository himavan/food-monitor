Introduction

This API is used for Food Monitoring Application.

Register User

HTTP Method: POST
HTTP Request: /api/monitor/
Description: To add new food monitored value.
Request Headers: 

                                        Content-Type : application/json

                                        Authentication Not Required
                                        
                                    

                                

                                
 Request Parameters:
                                    

                                        

                                        
                                        No request parameter required
                                        
                                    

                                

                                
 Request Body:
                                    

                                        

    { 
        "humidity":"String",
        "temperature":"String",
        "analog1":"String",
        "alcohol_ppm":"String",
        "analog2":"String",
        "methane_ppm": "String",
        "camera_state":"String"
    }

                                    

                                

                                
 Response:
                                    

                                        

    { 
        "humidity":"String",
        "temperature":"String",
        "analog1":"String",
